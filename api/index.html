API for publications processing results, uploads, processing catalyst cards and synthesis properties

Publication

Get publications list

Authorizations:
BearerAuth
query Parameters
search
string
Example: search=peng2022

search by publication name

page
integer >= 1
Default: 1
Example: page=1

page number (1-based)

page_size
integer [ 1 .. 100 ]
Default: 20
Example: page_size=20

number of items per page

Responses

https://catalysts.processing.com/api/publications
