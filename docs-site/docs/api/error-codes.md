# Коды ошибок

## HTTP-статусы

| Код | Статус | Когда возникает |
|---|---|---|
| 200 | OK | Успешное получение или обновление ресурса |
| 201 | Created | Успешное создание ресурса или загрузка публикации |
| 400 | Bad Request | Некорректный запрос или ошибка валидации |
| 401 | Unauthorized | Пользователь не авторизован |
| 404 | Not Found | Ресурс не найден |
| 500 | Internal Server Error | Внутренняя ошибка сервера |

## Коды ошибок в теле ответа

При ошибке API возвращает JSON с полями `error_code` и `message`

```json
{
  "error_code": "NOT_FOUND",
  "message": "Resource not found"
}
```

## Коды ошибок в теле ответа

| Код ошибки | HTTP | Описание |
|---|---:|---|
| `VALIDATION_ERROR` | 400 | Не переданы обязательные поля или нарушен формат данных |
| `INVALID_FILE_FORMAT` | 400 | Загруженный файл имеет неподдерживаемый формат |
| `UNAUTHORIZED` | 401 | Пользователь не авторизован |
| `PUBLICATION_NOT_FOUND` | 404 | Публикация с указанным `publication_id` не найдена |
| `CATALYST_NOT_FOUND` | 404 | Катализатор с указанным `catalyst_id` не найден |
| `SYNTHESIS_NOT_FOUND` | 404 | Синтез с указанным `synthesis_id` не найден |
| `PROCESSING_RESULT_NOT_FOUND` | 404 | Результат обработки публикации не найден |
| `PROCESSING_FAILED` | 500 | Ошибка при обработке публикации |
| `INTERNAL_ERROR` | 500 | Внутренняя ошибка сервера |
