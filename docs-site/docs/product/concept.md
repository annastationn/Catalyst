# Концепция продукта

## Общее описание

Система предназначена для автоматизированной обработки научных публикаций с целью извлечения структурированной информации о катализаторах и условиях синтеза.

Основная задача системы — преобразование неструктурированных данных (PDF, статьи) в формализованные записи, пригодные для дальнейшего анализа и хранения.

---

## Целевая аудитория

- Химики-исследователи
- Инженеры-технологи
- Руководители научных групп

---

## Ценность продукта

Система позволяет:
- сократить время анализа большого обьема научных данных
- автоматизировать процесс извлечения данных
- стандартизировать информацию о катализаторах
- повысить воспроизводимость исследований

---

## Основные функции

- загрузка научной публикации
- извлечение метаданных
- выделение информации о катализаторах
- извлечение условий синтеза
- хранение и просмотр результатов обработки

---

# Границы продукта

## Входит в систему

- обработка PDF-публикаций
- извлечение данных о катализаторах и синтезах
- хранение структурированных данных
- API для доступа к данным
- интерфейс для просмотра результатов

---

---

## Внешние системы

- хранилище файлов (Object Storage)
- базы данных (PostgreSQL)
- внешние источники публикаций 

---

# Ограничения

- точность извлечения зависит от качества публикации
- система не гарантирует 100% корректность данных
- поддерживаются только определенные форматы файлов (например PDF)
