# Catalyst Processing Documentation

Документация системы обработки научных публикаций и извлечения данных о катализаторах и синтезах

---

## О системе

Система предназначена для автоматической обработки научных публикаций с целью извлечения информации о катализаторах и параметрах их синтеза

Основные возможности:
- загрузка научных публикаций
- автоматическое извлечение данных (ML/NLP)
- формирование карточек катализаторов
- хранение и поиск данных
- просмотр результатов обработки

---

## Основные разделы документации

Документация разделена на логические блоки:

- Продукт — концепция, требования и границы системы  
- Анализ — бизнес-процессы (BPMN), Event Storming  
- Проектирование — UML-диаграммы  
- Интерфейс — макеты экранов и UX  
- Архитектура — структура системы и технологический стек  
- API — REST API и спецификация  

---

## Общий сценарий работы

1. Пользователь загружает публикацию  
2. Система ставит задачу в очередь  
3. Происходит обработка (ML/NLP, OCR)  
4. Формируются карточки катализаторов  
5. Результат сохраняется и доступен для просмотра
