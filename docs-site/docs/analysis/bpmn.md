# BPMN: Процесс обработки научной публикации

## Описание процесса

Процесс описывает полный цикл обработки научной публикации: от загрузки файла до формирования и сохранения карточки катализатора

---

## Основной сценарий

1. Публикация загружается в систему
2. Выполняется проверка файла и метаданных
3. Если файл некорректен — публикация отклоняется
4. Если файл корректен:
   - создается задача обработки
   - задача помещается в очередь
5. При наступлении очереди:
   - запускается обработка
6. Выполняется извлечение текста из публикации
7. Если текст не извлечен:
   - запускается OCR
   - при неудаче процесс завершается с ошибкой
8. После извлечения текста:
   - извлекаются параметры синтеза
   - формируется карточка катализатора
9. Карточка сохраняется
10. Если сохранение успешно:
    - записывается история обработки
    - процесс завершается

---

## Альтернативные сценарии

### Ошибка валидации
- файл отклоняется на этапе проверки

### Ошибка извлечения текста
- OCR не смог извлечь текст
- процесс завершается без результата

### Ошибка сохранения
- карточка не сохраняется
- процесс завершается с ошибкой
