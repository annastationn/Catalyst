# Макеты интерфейса

В рамках проектирования интерфейса были подготовлены макеты экранов системы обработки научных публикаций

## Главный экран

Главный экран является вводной страницей системы. Он содержит основные разделы: базу данных карточек катализаторов и архив загруженных публикаций.

![Главный экран](/img/main_desk.png)

На экране предусмотрены следующие элементы:

- кнопка меню в правом верхнем углу открывает боковое меню;
- кнопка загрузки публикации позволяет загрузить и обработать научную публикацию;
- строка поиска используется для поиска по системе;
- кнопка «База данных карточек» открывает список сохранённых карточек катализаторов;
- кнопка «Архив публикаций» открывает список публикаций, загруженных в систему.

## Экран загрузки файлов

Экран предназначен для загрузки и автоматической обработки научных публикаций с извлечением данных о катализаторах и синтезах.

![Экран загрузки файлов](/img/publication_upload.png)

После обработки публикации экран отображает результат: статус обработки, количество найденных синтезов и количество найденных катализаторов.

Основные элементы экрана:

- информационное окно с результатом обработки;
- кнопка «Катализатор» открывает сформированные карточки катализаторов;
- кнопка «Синтез» открывает найденный синтез, связанный с публикацией;
- строка ввода используется для уточняющих вопросов по результатам обработки.

## Экран с базой данных карточек катализаторов

Экран предназначен для просмотра и поиска сохранённых карточек катализаторов.

![Экран с базой данных карточек катализаторов](/img/db_cat.png)

Основные элементы экрана:

- поле поиска позволяет выполнять поиск по карточкам;
- список карточек отображает доступные катализаторы;
- при нажатии на карточку открывается подробная информация о катализаторе;
- кнопка «Добавить новую карточку» открывает создание новой карточки;
- кнопка «Редактировать список карточек» переводит интерфейс в режим редактирования списка.

## Экран взаимодействия с карточкой катализатора

Экран предназначен для просмотра и редактирования конкретной карточки катализатора.

![Экран взаимодействия с карточкой катализатора](/img/cat_card.png)

На экране отображаются:

- название катализатора;
- формула;
- параметры синтеза;
- изображение структуры;
- текстовое описание синтеза.

Иконка шестерёнки переводит карточку в режим редактирования.

## Экран с архивом публикаций

Экран предназначен для просмотра списка ранее загруженных публикаций.

![Экран с архивом публикаций](/img/db_pub.png)

Основные элементы экрана:

- поле поиска позволяет искать публикации по архиву;
- список публикаций отображает загруженные файлы;
- карточка публикации открывает экран просмотра выбранной публикации;
- кнопка «Редактировать архив публикаций» переводит интерфейс в режим редактирования.

## Экран выбранной публикации

Экран предназначен для просмотра содержимого выбранной статьи и связанных с ней сущностей.

![Экран выбранной публикации](/img/pub_card.png)

На экране отображаются:

- информационный блок публикации;
- текст публикации;
- связанные катализаторы;
- связанные синтезы;
- кнопка редактирования.

При нажатии на карточку катализатора открывается карточка катализатора, связанная с публикацией. При нажатии на карточку синтеза открывается информация о синтезе, извлечённом из статьи.

## Связь экранов с API

| Экран | Действие | Endpoint | Метод |
|---|---|---|---|
| Главный экран | Переход к архиву публикаций | `/api/publications` | GET |
| Главный экран | Переход к базе катализаторов | `/api/catalysts` | GET |
| Загрузка файлов | Загрузка публикации | `/api/publications` | POST |
| Загрузка файлов | Получение результата обработки | `/api/publications/{id}/result` | GET |
| Архив публикаций | Получение списка публикаций | `/api/publications` | GET |
| Архив публикаций | Поиск по публикациям | `/api/publications?search={query}` | GET |
| Выбранная публикация | Получение публикации | `/api/publications/{id}` | GET |
| Выбранная публикация | Получение результата обработки | `/api/publications/{id}/result` | GET |
| База карточек | Получение списка катализаторов | `/api/catalysts` | GET |
| База карточек | Поиск по катализаторам | `/api/catalysts?search={query}` | GET |
| Карточка катализатора | Получение карточки | `/api/catalysts/{id}` | GET |
| Карточка катализатора | Редактирование карточки | `/api/catalysts/{id}` | PATCH |
| Выбранная публикация | Получение синтеза | `/api/syntheses/{id}` | GET |
| База карточек | Создание новой карточки | `/api/catalysts` | POST |

## Итог

Макеты интерфейса описывают основные пользовательские сценарии системы: загрузку публикации, просмотр результата обработки, работу с архивом публикаций и карточками катализаторов. На основе этих экранов были уточнены необходимые REST API endpoints.
